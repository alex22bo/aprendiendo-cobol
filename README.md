# Aprendiendo COBOL — Bitácora y Portfolio

Repositorio de autoentrenamiento en COBOL y ecosistema mainframe (z/OS), documentando el progreso desde una base previa en C, C++ y Python.

## Objetivo

Alcanzar nivel junior/semi-senior empleable en COBOL + JCL + CICS + DB2 en un plazo de 4-6 meses, con evidencia práctica verificable (no solo teoría).
